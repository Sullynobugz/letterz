# Assets Directory

This directory should contain the following image assets for the app:

- `icon.png` (1024x1024) - App icon
- `splash.png` (1242x2436 or similar) - Splash screen
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (48x48 or larger) - Web favicon

For development, you can use placeholder images or generate them using:
- https://www.appicon.co/ (for app icons)
- Simple solid color images
- Or run: `npx expo prebuild` which may generate defaults

The app will work without these assets but Expo may show warnings.
